# Prompts - Industria Alimenticia

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Industria Alimenticia.

## Base do especialista

- Termos do mercado: equipamento em estoque, manutencao tecnica, pecas de reposicao, maquina para producao, ferramenta profissional, seguranca de operacao.
- Dores principais: maquina parada gera prejuizo; cliente precisa de peca correta; duvida sobre compatibilidade; prazo de entrega impacta producao.
- Ofertas principais: manutencao de equipamento; pecas de reposicao; orcamento tecnico por modelo; kit de ferramentas.
- CTAs principais: Envie o modelo da maquina; Consulte peca compativel; Agende manutencao tecnica; Verifique pronta entrega.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
