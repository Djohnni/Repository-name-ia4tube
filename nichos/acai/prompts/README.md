# Prompts - Acai

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Acai.

## Base do especialista

- Termos do mercado: cardapio do dia, pedido pelo WhatsApp, delivery local, combo promocional, prato destaque, ingredientes frescos.
- Dores principais: movimento fraco fora do horario de pico; cliente nao sabe o cardapio; baixa venda no delivery; produto fresco precisa girar.
- Ofertas principais: combo da semana; prato do dia; pedido para retirada; delivery local.
- CTAs principais: Peca pelo WhatsApp; Veja o cardapio de hoje; Reserve seu pedido; Garanta seu combo.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
