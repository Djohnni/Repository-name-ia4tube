# Promocao - Capas Para Celular

A promocao deve destacar uma oferta real de Capas Para Celular.

Exemplos seguros:

- novidades da semana.
- item destaque.
- kit presente.
- queima de estoque.
- colecao nova.
- combo de itens.

Nao inventar desconto, preco, brinde, prazo ou condicao nao informada.
