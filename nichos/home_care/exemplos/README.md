# Exemplos - Home Care

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Home Care.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com ambiente clinico acolhedor, tons claros, pessoas reais, sensacao de seguranca e cuidado.
- Pouco texto e leitura rapida.
