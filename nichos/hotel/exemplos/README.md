# Exemplos - Hotel

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Hotel.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com estrada, veiculo, quarto, mala, mapa, destino ou equipe em movimento com CTA de reserva.
- Pouco texto e leitura rapida.
