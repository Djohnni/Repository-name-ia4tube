# Prompts - Clinica Odontologica

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Clinica Odontologica.

## Base do especialista

- Termos do mercado: avaliacao odontologica, limpeza dental, clareamento dental, implantes, ortodontia, alinhadores.
- Dores principais: medo de dor no tratamento; vergonha do sorriso; dente quebrado ou ausente; manchas nos dentes.
- Ofertas principais: avaliacao odontologica; limpeza dental preventiva; clareamento com avaliacao; consulta para implantes.
- CTAs principais: Agende sua avaliacao odontologica; Marque sua consulta; Cuide do seu sorriso; Tire duvidas sobre seu tratamento.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
