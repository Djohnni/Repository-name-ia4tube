# Prompts - Clinica Estetica

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Clinica Estetica.

## Base do especialista

- Termos do mercado: avaliacao profissional, consulta agendada, triagem de sintomas, acompanhamento de rotina, prevencao e cuidado, retorno planejado.
- Dores principais: demora para marcar atendimento; medo de piorar o quadro; duvida sobre qual especialista procurar; falta de acompanhamento regular.
- Ofertas principais: agenda de consultas; avaliacao inicial; check-up preventivo; acompanhamento individual.
- CTAs principais: Agende sua consulta; Marque sua avaliacao; Cuide da sua saude hoje; Verifique horarios disponiveis.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
