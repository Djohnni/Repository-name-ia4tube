# Exemplos - Loja Geek

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Loja Geek.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com item em destaque, vitrine limpa, etiquetas discretas, cores da marca, CTA para reserva ou compra.
- Pouco texto e leitura rapida.
