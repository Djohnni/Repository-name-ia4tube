# Prompts - Tintas

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Tintas.

## Base do especialista

- Termos do mercado: Tintas local, agenda de Tintas, avaliacao de Tintas, servico de Tintas, prova visual de Tintas, processo de Tintas.
- Dores principais: cliente nao entende quando procurar Tintas; baixa lembranca da marca de Tintas; duvida sobre escopo de Tintas; comparacao por preco em Tintas.
- Ofertas principais: avaliacao inicial de Tintas; agenda aberta para Tintas; pacote de Tintas conforme necessidade; servico de Tintas com prazo combinado.
- CTAs principais: Chame para avaliar Tintas; Envie sua necessidade de Tintas; Veja opcoes de Tintas; Reserve horario para Tintas.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
