# Prompts - Produtos De Limpeza

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Produtos De Limpeza.

## Base do especialista

- Termos do mercado: limpeza programada, higienizacao profunda, coleta ou retirada agendada, controle de pragas, manutencao preventiva, equipe no local.
- Dores principais: ambiente com sujeira dificil; mau cheiro ou risco sanitario; falta de tempo para limpeza pesada; cliente precisa de agenda recorrente.
- Ofertas principais: limpeza pesada agendada; higienizacao de ambiente; contrato recorrente; controle preventivo.
- CTAs principais: Agende sua limpeza; Envie fotos do local; Cuide do seu ambiente; Reserve equipe para sua empresa.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
