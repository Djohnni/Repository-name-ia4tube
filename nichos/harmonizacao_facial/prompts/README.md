# Prompts - Harmonizacao Facial

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Harmonizacao Facial.

## Base do especialista

- Termos do mercado: agenda de horarios, protocolo de beleza, resultado natural, manutencao periodica, procedimento estetico, autocuidado.
- Dores principais: falta de horario antes de evento; medo de resultado artificial; visual sem manutencao; cliente esquece retoque.
- Ofertas principais: agenda da semana de beleza; pacote de manutencao; protocolo de beleza; preparacao para evento.
- CTAs principais: Agende seu horario; Reserve seu cuidado; Veja horarios da semana; Realce sua beleza.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
