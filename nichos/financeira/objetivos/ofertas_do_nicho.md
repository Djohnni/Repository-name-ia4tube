# Ofertas do Nicho - Financeira

## Ofertas especificas

- analise inicial do caso.
- consultoria por area.
- regularizacao documental.
- planejamento tributario ou contratual.
- revisao de documentos.
- atendimento para empresas.
- orientacao para pessoa fisica.
- agenda de consultas.
- avaliacao inicial de Financeira.
- agenda aberta para Financeira.
- pacote de Financeira conforme necessidade.
- servico de Financeira com prazo combinado.

## Como usar

- A oferta deve estar ligada a uma dor real de Financeira.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
