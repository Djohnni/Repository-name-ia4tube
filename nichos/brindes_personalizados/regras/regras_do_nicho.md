# Regras do Nicho - Brindes Personalizados

- Nunca inventar preco, prazo, garantia ou disponibilidade para Brindes Personalizados.
- Usar dados reais do cliente antes de qualquer orientacao do nicho.
- Evitar frases vagas; toda arte deve mencionar Brindes Personalizados, oferta, dor ou CTA especifico.
- Nao misturar exemplos de outros mercados.
- Manter uma mensagem principal curta e facil de entender.
- Usar prova visual autorizada quando existir.
- A arte deve parecer de Brindes Personalizados em texto, imagem, oferta e CTA.
