# Saidas Esperadas

Futuramente, o pipeline pode produzir:

- prompt final para imagem;
- texto principal da arte;
- subtitulo ou apoio;
- CTA;
- legenda para postagem;
- checklist de validacao;
- avisos ao usuario sobre dados ausentes;
- sugestoes de campanha.

As saidas devem respeitar dados reais do pedido e nao inventar informacoes comerciais.
