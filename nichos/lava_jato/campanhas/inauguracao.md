# Campanha: Inauguracao

## Quando usar

Para abertura de novo lava jato, novo endereco, nova equipe ou nova fase.

## Mensagem central

Novo espaco para cuidar do seu carro com qualidade.

## Informacoes importantes

- nome do lava jato;
- data, se informada;
- endereco, se informado;
- servicos principais;
- CTA para conhecer ou agendar.

## Erros a evitar

Nao inventar data, endereco ou promocao de inauguracao.
