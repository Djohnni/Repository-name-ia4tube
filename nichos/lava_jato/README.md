# Nicho: lava_jato

Pasta de conhecimento do nicho individual `lava_jato`.

Este nicho organiza inteligencia comercial, criativa e operacional para criar artes, textos, campanhas e validacoes voltadas a lava jatos, esteticas automotivas e servicos de limpeza automotiva.

Esta pasta ainda nao esta conectada ao runtime. Ela serve como base documental para evolucao futura do Motor de Nichos.

## Escopo

- Lavagem simples e completa.
- Higienizacao interna.
- Polimento, cristalizacao e vitrificacao, quando oferecidos.
- Limpeza de bancos, carpetes, teto e painel.
- Lavagem de motos, carros, SUVs e frotas.
- Promocoes, agenda aberta e campanhas locais.
- Artes de antes e depois.

## Uso futuro

No futuro, o Motor de Nichos podera carregar `KNOWLEDGE_SUMMARY.md` e outros arquivos desta pasta para orientar prompts, legendas, CTAs e validacoes especificas para lava jatos.
