# Prompt: Validar Arte Para Lava Jato

Valide se a arte esta adequada para vender servicos de lava jato.

Checklist:

- O servico principal esta claro
- Existe CTA visivel
- A arte transmite limpeza, brilho ou cuidado
- O texto e curto e facil de ler
- O visual combina com lava jato/estetica automotiva
- O logo foi respeitado
- Nao foram inventados precos ou promessas
- A campanha ajuda o cliente a agir agora
