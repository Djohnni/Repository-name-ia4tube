# Prompt: Criar Arte Para Lava Jato

Crie uma arte comercial para um lava jato ou estetica automotiva.

Priorize:

- beneficio visual claro;
- limpeza, brilho e cuidado;
- CTA forte para WhatsApp ou agendamento;
- texto curto e legivel;
- respeito ao nome, logo e informacoes do cliente;
- tom coerente com o servico: popular, rapido, detalhado ou premium.

Nunca invente preco, endereco, telefone, produto tecnico ou garantia.

Use os dados do pedido como fonte principal.
