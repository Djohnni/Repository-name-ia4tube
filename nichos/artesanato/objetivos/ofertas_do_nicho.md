# Ofertas do Nicho - Artesanato

## Ofertas especificas

- avaliacao inicial de Artesanato.
- agenda aberta para Artesanato.
- pacote de Artesanato conforme necessidade.
- servico de Artesanato com prazo combinado.
- campanha local de Artesanato.

## Como usar

- A oferta deve estar ligada a uma dor real de Artesanato.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
