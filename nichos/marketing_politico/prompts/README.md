# Prompts - Marketing Politico

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Marketing Politico.

## Base do especialista

- Termos do mercado: identidade visual, criativo para redes sociais, landing page, site responsivo, trafego pago, briefing do projeto.
- Dores principais: marca parece amadora; cliente nao sabe o que postar; baixa conversao no site; falta de criativos para campanha.
- Ofertas principais: pacote de artes mensais; criacao de identidade visual; site institucional; landing page de captacao.
- CTAs principais: Solicite um briefing; Veja exemplos de projetos; Comece sua campanha; Crie sua identidade visual.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
