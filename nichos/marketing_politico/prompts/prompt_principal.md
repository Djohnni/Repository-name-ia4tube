# Prompt Principal - Marketing Politico

Crie uma arte para Marketing Politico usando termos, dores e CTAs especificos deste mercado.

Use como base:

- Termos: identidade visual, criativo para redes sociais, landing page, site responsivo, trafego pago, briefing do projeto, arquivo para impressao, prova digital.
- Dores: marca parece amadora; cliente nao sabe o que postar; baixa conversao no site; falta de criativos para campanha; briefing incompleto atrasa entrega.
- Ofertas: pacote de artes mensais; criacao de identidade visual; site institucional; landing page de captacao; campanha de trafego pago.
- CTAs: Solicite um briefing; Veja exemplos de projetos; Comece sua campanha; Crie sua identidade visual; Peca seu material grafico.

Nao invente preco, prazo, resultado ou garantia. Priorize os dados reais do cliente.
