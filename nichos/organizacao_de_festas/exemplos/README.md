# Exemplos - Organizacao De Festas

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Organizacao De Festas.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com festa real, mesa montada, luz, decoracao, pessoas felizes, chamada para reserva de data.
- Pouco texto e leitura rapida.
