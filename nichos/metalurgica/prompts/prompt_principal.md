# Prompt Principal - Metalurgica

Crie uma arte para Metalurgica usando termos, dores e CTAs especificos deste mercado.

Use como base:

- Termos: equipamento em estoque, manutencao tecnica, pecas de reposicao, maquina para producao, ferramenta profissional, seguranca de operacao, instalacao tecnica, assistencia no equipamento.
- Dores: maquina parada gera prejuizo; cliente precisa de peca correta; duvida sobre compatibilidade; prazo de entrega impacta producao; falta de assistencia tecnica.
- Ofertas: manutencao de equipamento; pecas de reposicao; orcamento tecnico por modelo; kit de ferramentas; equipamento para producao.
- CTAs: Envie o modelo da maquina; Consulte peca compativel; Agende manutencao tecnica; Verifique pronta entrega; Peca orientacao por foto.

Nao invente preco, prazo, resultado ou garantia. Priorize os dados reais do cliente.
