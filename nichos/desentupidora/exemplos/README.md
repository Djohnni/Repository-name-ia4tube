# Exemplos - Desentupidora

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Desentupidora.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com ambiente limpo, equipe equipada, antes e depois autorizado, brilho, organizacao e seguranca.
- Pouco texto e leitura rapida.
