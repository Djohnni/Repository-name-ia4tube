# Regras do Nicho - Visual Merchandising

- Nunca inventar preco, prazo, garantia ou disponibilidade para Visual Merchandising.
- Usar dados reais do cliente antes de qualquer orientacao do nicho.
- Evitar frases vagas; toda arte deve mencionar Visual Merchandising, oferta, dor ou CTA especifico.
- Nao misturar exemplos de outros mercados.
- Manter uma mensagem principal curta e facil de entender.
- Usar prova visual autorizada quando existir.
- A arte deve parecer de Visual Merchandising em texto, imagem, oferta e CTA.
