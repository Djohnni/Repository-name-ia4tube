# Prompts - Tapecaria Automotiva

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Tapecaria Automotiva.

## Base do especialista

- Termos do mercado: diagnostico automotivo, revisao preventiva, troca de oleo, freios e suspensao, scanner automotivo, alinhamento e balanceamento.
- Dores principais: medo de ficar parado na rua; barulho sem origem no carro; manutencao atrasada; gasto alto de combustivel.
- Ofertas principais: check-up automotivo; revisao antes de viagem; troca de oleo com filtro; diagnostico por scanner.
- CTAs principais: Agende sua revisao; Verifique seu carro hoje; Chame para diagnostico; Cuide dos freios antes da viagem.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
