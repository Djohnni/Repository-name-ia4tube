# Exemplos - Retifica De Motores

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Retifica De Motores.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com carro, moto, ferramenta, elevador automotivo, brilho, estrada ou detalhe mecanico com CTA visivel.
- Pouco texto e leitura rapida.
