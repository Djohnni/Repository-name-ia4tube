# Prompt Principal - Loja De Maquiagem

Crie uma arte para Loja De Maquiagem usando termos, dores e CTAs especificos deste mercado.

Use como base:

- Termos: agenda de horarios, protocolo de beleza, resultado natural, manutencao periodica, procedimento estetico, autocuidado, transformacao visual autorizada, preparacao para evento.
- Dores: falta de horario antes de evento; medo de resultado artificial; visual sem manutencao; cliente esquece retoque; duvida sobre procedimento ideal.
- Ofertas: agenda da semana de beleza; pacote de manutencao; protocolo de beleza; preparacao para evento; avaliacao estetica.
- CTAs: Agende seu horario; Reserve seu cuidado; Veja horarios da semana; Realce sua beleza; Chame para avaliar seu caso.

Nao invente preco, prazo, resultado ou garantia. Priorize os dados reais do cliente.
