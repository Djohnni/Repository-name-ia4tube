# Ofertas do Nicho - Lareiras

## Ofertas especificas

- avaliacao inicial de Lareiras.
- agenda aberta para Lareiras.
- pacote de Lareiras conforme necessidade.
- servico de Lareiras com prazo combinado.
- campanha local de Lareiras.
- orientacao antes de contratar Lareiras.
- retorno para clientes antigos de Lareiras.
- divulgacao de vagas para Lareiras.

## Como usar

- A oferta deve estar ligada a uma dor real de Lareiras.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
