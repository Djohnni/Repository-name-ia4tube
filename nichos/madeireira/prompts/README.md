# Prompts - Madeireira

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Madeireira.

## Base do especialista

- Termos do mercado: visita tecnica, medicao no local, projeto sob medida, instalacao com acabamento, orcamento por ambiente, manutencao preventiva.
- Dores principais: medo de atraso na obra; receio de servico mal acabado; orcamento pouco claro; urgencia de reparo em casa.
- Ofertas principais: visita tecnica agendada; orcamento por medida; projeto sob medida; manutencao preventiva.
- CTAs principais: Agende uma visita tecnica; Envie medidas ou fotos; Avalie seu ambiente; Planeje sua reforma.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
