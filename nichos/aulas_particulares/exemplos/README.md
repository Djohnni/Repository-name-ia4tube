# Exemplos - Aulas Particulares

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Aulas Particulares.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com alunos, sala, professor, material didatico, ambiente seguro e organizado.
- Pouco texto e leitura rapida.
