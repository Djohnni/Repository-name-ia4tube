# Textos para Arte - Web Design

## Chamadas curtas

- pacote de artes mensais: Solicite um briefing.
- criacao de identidade visual: Veja exemplos de projetos.
- site institucional: Comece sua campanha.
- landing page de captacao: Crie sua identidade visual.
- campanha de trafego pago: Peca seu material grafico.
- material impresso: Transforme sua presenca digital.

## Beneficios

- Resolva: marca parece amadora.
- Resolva: cliente nao sabe o que postar.
- Resolva: baixa conversao no site.
- Resolva: falta de criativos para campanha.
- Resolva: briefing incompleto atrasa entrega.
- Resolva: arquivo errado para impressao.

## Evitar

- nao usar exemplo de outro mercado.
- nao inventar preco, prazo ou garantia.
- nao prometer resultado garantido.
- nao esconder a oferta principal.
- nao usar frase vaga sem mostrar Web Design.
- nao prometer resultado que depende de avaliacao.
- nao misturar exemplos de outros mercados.
