# Datas Comemorativas - Web Design

## Datas comerciais adaptaveis

- Dia das Maes: conectar Web Design a cuidado, presente, familia ou praticidade quando fizer sentido.
- Dia dos Pais: usar praticidade, protecao, cuidado ou experiencia ligada a Web Design.
- Black Friday: divulgar pacotes, agenda, kits ou itens de maior procura em Web Design.
- Natal e fim de ano: reforcar preparacao, presentes, revisao, agenda ou planejamento.
- Volta as aulas: aplicar quando Web Design atender familia, rotina, educacao, casa ou organizacao.
- Ferias e feriados: usar gancho de viagem, casa, lazer, manutencao ou movimento local conforme o nicho.

## Regra

A data deve ser gancho para uma necessidade real de Web Design, nunca um tema solto.
