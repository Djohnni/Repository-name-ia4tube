# Regras do Nicho - Web Design

- Nunca inventar preco, prazo, garantia ou disponibilidade para Web Design.
- Usar dados reais do cliente antes de qualquer orientacao do nicho.
- Evitar frases vagas; toda arte deve mencionar Web Design, oferta, dor ou CTA especifico.
- Nao misturar exemplos de outros mercados.
- Manter uma mensagem principal curta e facil de entender.
- Usar prova visual autorizada quando existir.
- A arte deve parecer de Web Design em texto, imagem, oferta e CTA.
