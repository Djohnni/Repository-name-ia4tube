# Exemplos - Protese Dentaria

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Protese Dentaria.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com sorriso natural, consultorio claro, jaleco, tons azul branco ou verde, cuidado e acolhimento.
- Pouco texto e leitura rapida.
