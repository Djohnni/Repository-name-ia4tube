# Exemplos - Consultoria Financeira

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Consultoria Financeira.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com mesa de trabalho, documentos, predios, assinatura, equipe consultiva, tom sobrio e confiavel.
- Pouco texto e leitura rapida.
