# Ofertas do Nicho - Bazar

## Ofertas especificas

- novidades da semana.
- item destaque.
- kit presente.
- queima de estoque.
- colecao nova.
- combo de itens.
- promocao por categoria.
- vitrine de lancamentos.
- avaliacao inicial de Bazar.
- agenda aberta para Bazar.
- pacote de Bazar conforme necessidade.
- servico de Bazar com prazo combinado.

## Como usar

- A oferta deve estar ligada a uma dor real de Bazar.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
