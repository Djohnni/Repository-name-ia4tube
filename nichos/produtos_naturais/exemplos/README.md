# Exemplos - Produtos Naturais

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Produtos Naturais.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com imagem real ou simbolo claro de Produtos Naturais, composicao limpa, beneficio em destaque e CTA visivel.
- Pouco texto e leitura rapida.
