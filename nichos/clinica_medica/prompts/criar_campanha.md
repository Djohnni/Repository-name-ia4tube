# Criar Campanha - Clinica Medica

Campanhas recomendadas:

- campanha de prevencao.
- agenda de consultas da semana.
- mes de conscientizacao.
- check-up de rotina.
- volta as atividades com cuidado.
- campanha para novos pacientes.
- orientacao sobre sintomas.
- cuidado antes de feriados.
- agenda aberta de Clinica Medica.
- semana de divulgacao de Clinica Medica.
- campanha para clientes do bairro.
- campanha educativa sobre Clinica Medica.
