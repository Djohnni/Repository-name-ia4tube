# Regras do Nicho - Clinica Medica

- Nunca inventar preco, prazo, garantia ou disponibilidade para Clinica Medica.
- Usar dados reais do cliente antes de qualquer orientacao do nicho.
- Evitar frases vagas; toda arte deve mencionar Clinica Medica, oferta, dor ou CTA especifico.
- Nao misturar exemplos de outros mercados.
- Manter uma mensagem principal curta e facil de entender.
- Usar prova visual autorizada quando existir.
- A arte deve parecer de Clinica Medica em texto, imagem, oferta e CTA.
