# Prompt Principal - Supermercado De Bairro

Crie uma arte para Supermercado De Bairro usando termos, dores e CTAs especificos deste mercado.

Use como base:

- Termos: cardapio do dia, pedido pelo WhatsApp, delivery local, combo promocional, prato destaque, ingredientes frescos, encomendas para eventos, reserva quando houver.
- Dores: movimento fraco fora do horario de pico; cliente nao sabe o cardapio; baixa venda no delivery; produto fresco precisa girar; concorrencia por preco no aplicativo.
- Ofertas: combo da semana; prato do dia; pedido para retirada; delivery local; encomendas para festas.
- CTAs: Peca pelo WhatsApp; Veja o cardapio de hoje; Reserve seu pedido; Garanta seu combo; Chame para encomendar.

Nao invente preco, prazo, resultado ou garantia. Priorize os dados reais do cliente.
