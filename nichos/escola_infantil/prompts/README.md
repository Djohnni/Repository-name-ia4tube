# Prompts - Escola Infantil

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Escola Infantil.

## Base do especialista

- Termos do mercado: matriculas abertas, turma nova, aula experimental, certificado quando houver, metodologia de ensino, horarios de aula.
- Dores principais: pais procuram seguranca para matricular; aluno tem dificuldade de acompanhar conteudo; turmas com vagas abertas; baixa procura fora do inicio do semestre.
- Ofertas principais: matriculas abertas; aula experimental; turma iniciante; pacote mensal de aulas.
- CTAs principais: Matricule-se agora; Agende uma aula experimental; Conheca a metodologia; Reserve sua vaga na turma.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
