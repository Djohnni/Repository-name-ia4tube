# Ofertas do Nicho - Bronzeamento

## Ofertas especificas

- agenda da semana de beleza.
- pacote de manutencao.
- protocolo de beleza.
- preparacao para evento.
- avaliacao estetica.
- combo de cuidados.
- retoque programado.
- campanha de autoestima.
- avaliacao inicial de Bronzeamento.
- agenda aberta para Bronzeamento.
- pacote de Bronzeamento conforme necessidade.
- servico de Bronzeamento com prazo combinado.

## Como usar

- A oferta deve estar ligada a uma dor real de Bronzeamento.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
