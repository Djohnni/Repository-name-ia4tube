# Ofertas do Nicho - Brecho

## Ofertas especificas

- novidades da semana.
- item destaque.
- kit presente.
- queima de estoque.
- colecao nova.
- combo de itens.
- promocao por categoria.
- vitrine de lancamentos.
- avaliacao inicial de Brecho.
- agenda aberta para Brecho.
- pacote de Brecho conforme necessidade.
- servico de Brecho com prazo combinado.

## Como usar

- A oferta deve estar ligada a uma dor real de Brecho.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
