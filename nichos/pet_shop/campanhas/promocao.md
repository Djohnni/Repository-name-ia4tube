# Campanha: Promocao - Pet Shop

## Ofertas promissoras

- banho + tosa higienica.
- pacote mensal de banho.
- hidratacao para pelos.
- combo banho + unhas.
- taxi dog promocional.

Servico especifico + beneficio claro + condicao + CTA.

## Exemplo

banho + tosa higienica com chamada para Agende o banho do seu pet.

## Cuidado

Promocao sem item concreto parece generica. Sempre dizer o que esta sendo promovido.
