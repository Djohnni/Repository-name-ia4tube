# Prompt Principal do Nicho

Use este arquivo como rascunho central do prompt especializado do nicho.

Ele deve definir:

- papel da IA para este nicho;
- tipo de arte ou campanha que deve gerar;
- publico atendido;
- tom de comunicacao;
- regras obrigatorias;
- limites do que nao pode inventar;
- criterios de qualidade.

Este prompt nao substitui prompts de producao automaticamente.
