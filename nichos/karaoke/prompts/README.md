# Prompts - Karaoke

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Karaoke.

## Base do especialista

- Termos do mercado: reserva de data, montagem do evento, pacote de festa, decoracao tematica, lista de convidados, buffet para evento.
- Dores principais: data desejada lota rapido; medo de festa desorganizada; orcamento precisa caber no bolso; duvida sobre quantidade de convidados.
- Ofertas principais: pacote de festa completo; reserva de data; decoracao personalizada; combo som e iluminacao.
- CTAs principais: Reserve sua data; Monte sua festa conosco; Veja pacotes de evento; Garanta sua data especial.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
