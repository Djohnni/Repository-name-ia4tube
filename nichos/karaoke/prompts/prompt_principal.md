# Prompt Principal - Karaoke

Crie uma arte para Karaoke usando termos, dores e CTAs especificos deste mercado.

Use como base:

- Termos: reserva de data, montagem do evento, pacote de festa, decoracao tematica, lista de convidados, buffet para evento, som e iluminacao, animacao infantil.
- Dores: data desejada lota rapido; medo de festa desorganizada; orcamento precisa caber no bolso; duvida sobre quantidade de convidados; falta de tempo para organizar.
- Ofertas: pacote de festa completo; reserva de data; decoracao personalizada; combo som e iluminacao; buffet para convidados.
- CTAs: Reserve sua data; Monte sua festa conosco; Veja pacotes de evento; Garanta sua data especial; Chame para planejar sua festa.

Nao invente preco, prazo, resultado ou garantia. Priorize os dados reais do cliente.
