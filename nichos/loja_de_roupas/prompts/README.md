# Prompts - Loja De Roupas

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Loja De Roupas.

## Base do especialista

- Termos do mercado: estoque disponivel, novidades da semana, vitrine promocional, colecao nova, item mais procurado, tamanhos e cores.
- Dores principais: estoque parado; cliente nao sabe novidades; concorrencia com marketplace; duvida sobre tamanho ou modelo.
- Ofertas principais: novidades da semana; item destaque; kit presente; queima de estoque.
- CTAs principais: Veja novidades da loja; Consulte disponibilidade; Reserve seu item; Garanta antes que acabe.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
