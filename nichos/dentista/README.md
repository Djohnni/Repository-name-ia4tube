# Nicho: dentista

Pasta de conhecimento do nicho individual `dentista`.

Este nicho organiza inteligencia comercial, criativa e operacional para criar artes, textos, campanhas e validacoes voltadas a dentistas, clinicas odontologicas, consultorios e servicos de estetica dental.

Esta pasta ainda nao esta conectada ao runtime. Ela serve como base documental para evolucao futura do Motor de Nichos.

## Escopo

- Avaliacao odontologica.
- Limpeza dental/profilaxia.
- Clareamento dental.
- Implantes.
- Ortodontia/aparelho.
- Lentes de contato dental, quando informado.
- Harmonizacao relacionada ao sorriso, quando oferecida.
- Agenda aberta.
- Campanhas de cuidado, saude, estetica e prevencao.

## Uso futuro

No futuro, o Motor de Nichos podera carregar `KNOWLEDGE_SUMMARY.md` e outros arquivos desta pasta para orientar prompts, legendas, CTAs e validacoes especificas para dentistas.
