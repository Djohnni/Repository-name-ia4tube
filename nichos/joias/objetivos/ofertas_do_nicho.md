# Ofertas do Nicho - Joias

## Ofertas especificas

- novidades da semana.
- item destaque.
- kit presente.
- queima de estoque.
- colecao nova.
- combo de itens.
- promocao por categoria.
- vitrine de lancamentos.
- avaliacao inicial de Joias.
- agenda aberta para Joias.
- pacote de Joias conforme necessidade.
- servico de Joias com prazo combinado.

## Como usar

- A oferta deve estar ligada a uma dor real de Joias.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
