# Regras do Nicho - Doces Personalizados

- Nunca inventar preco, prazo, garantia ou disponibilidade para Doces Personalizados.
- Usar dados reais do cliente antes de qualquer orientacao do nicho.
- Evitar frases vagas; toda arte deve mencionar Doces Personalizados, oferta, dor ou CTA especifico.
- Nao misturar exemplos de outros mercados.
- Manter uma mensagem principal curta e facil de entender.
- Usar prova visual autorizada quando existir.
- A arte deve parecer de Doces Personalizados em texto, imagem, oferta e CTA.
