# Prompt Principal - Cursos Online

Crie uma arte para Cursos Online usando termos, dores e CTAs especificos deste mercado.

Use como base:

- Termos: matriculas abertas, turma nova, aula experimental, certificado quando houver, metodologia de ensino, horarios de aula, aprendizado pratico, acompanhamento pedagogico.
- Dores: pais procuram seguranca para matricular; aluno tem dificuldade de acompanhar conteudo; turmas com vagas abertas; baixa procura fora do inicio do semestre; duvida sobre metodologia.
- Ofertas: matriculas abertas; aula experimental; turma iniciante; pacote mensal de aulas; preparacao para prova.
- CTAs: Matricule-se agora; Agende uma aula experimental; Conheca a metodologia; Reserve sua vaga na turma; Chame para ver horarios.

Nao invente preco, prazo, resultado ou garantia. Priorize os dados reais do cliente.
