# Ofertas do Nicho - Coworking

## Ofertas especificas

- avaliacao inicial de Coworking.
- agenda aberta para Coworking.
- pacote de Coworking conforme necessidade.
- servico de Coworking com prazo combinado.
- campanha local de Coworking.
- orientacao antes de contratar Coworking.
- retorno para clientes antigos de Coworking.
- divulgacao de vagas para Coworking.

## Como usar

- A oferta deve estar ligada a uma dor real de Coworking.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
