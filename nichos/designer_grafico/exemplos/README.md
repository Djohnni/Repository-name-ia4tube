# Exemplos - Designer Grafico

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Designer Grafico.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com mockups, telas, impressos, grid criativo, contraste moderno, prova de portfolio e CTA profissional.
- Pouco texto e leitura rapida.
