# Prompts - Turismo Rural

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Turismo Rural.

## Base do especialista

- Termos do mercado: reserva antecipada, disponibilidade de datas, rota programada, conforto na viagem, bagagem ou carga segura, retirada e entrega.
- Dores principais: medo de atraso; duvida sobre disponibilidade; preocupacao com seguranca da carga; falta de informacao sobre roteiro.
- Ofertas principais: reserva de data; pacote de viagem; frete programado; transfer agendado.
- CTAs principais: Reserve sua data; Consulte disponibilidade; Planeje sua viagem; Agende seu transporte.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
