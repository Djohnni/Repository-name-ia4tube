# Exemplos - Consorcio

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Consorcio.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com imagem real ou simbolo claro de Consorcio, composicao limpa, beneficio em destaque e CTA visivel.
- Pouco texto e leitura rapida.
