# Regras do Nicho - Itens Personalizados

- Nunca inventar preco, prazo, garantia ou disponibilidade para Itens Personalizados.
- Usar dados reais do cliente antes de qualquer orientacao do nicho.
- Evitar frases vagas; toda arte deve mencionar Itens Personalizados, oferta, dor ou CTA especifico.
- Nao misturar exemplos de outros mercados.
- Manter uma mensagem principal curta e facil de entender.
- Usar prova visual autorizada quando existir.
- A arte deve parecer de Itens Personalizados em texto, imagem, oferta e CTA.
