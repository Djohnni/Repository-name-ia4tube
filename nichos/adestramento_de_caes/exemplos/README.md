# Exemplos - Adestramento De Caes

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Adestramento De Caes.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com pet limpo, tutor feliz, banho, tosa, racao ou acessorios com tom carinhoso e CTA claro.
- Pouco texto e leitura rapida.
