# Prompts - Assessoria Contabil

Esta pasta guarda instrucoes de prompt para criar, validar, legendar e melhorar artes de Assessoria Contabil.

## Base do especialista

- Termos do mercado: analise do caso, documentos necessarios, consultoria especializada, planejamento do processo, prazo legal ou administrativo, regularizacao documental.
- Dores principais: medo de multa ou prejuizo; burocracia dificil de entender; documentos pendentes; prazo apertado.
- Ofertas principais: analise inicial do caso; consultoria por area; regularizacao documental; planejamento tributario ou contratual.
- CTAs principais: Agende uma analise; Envie sua duvida com seguranca; Organize seus documentos; Fale sobre seu caso.

## Regra

O prompt deve priorizar dados reais do cliente e usar o conhecimento do nicho apenas como apoio estrategico.
