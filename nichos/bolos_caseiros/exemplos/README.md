# Exemplos - Bolos Caseiros

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Bolos Caseiros.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com foto real e apetitosa, close no produto, cores quentes, preco apenas se informado, CTA para pedido.
- Pouco texto e leitura rapida.
