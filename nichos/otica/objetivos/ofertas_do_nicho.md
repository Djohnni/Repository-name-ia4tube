# Ofertas do Nicho - Otica

## Ofertas especificas

- novidades da semana.
- item destaque.
- kit presente.
- queima de estoque.
- colecao nova.
- combo de itens.
- promocao por categoria.
- vitrine de lancamentos.
- avaliacao inicial de Otica.
- agenda aberta para Otica.
- pacote de Otica conforme necessidade.
- servico de Otica com prazo combinado.

## Como usar

- A oferta deve estar ligada a uma dor real de Otica.
- Se houver preco, desconto ou prazo, usar somente quando informado pelo cliente.
- Priorizar ofertas que gerem resposta rapida no WhatsApp.
