# Exemplos - Conserto De Televisao

Esta pasta guarda exemplos de campanhas, CTAs e criativos que fazem sentido para Conserto De Televisao.

## Bons exemplos devem conter

- Dor real do mercado.
- Oferta especifica.
- CTA acionavel.
- Imagem coerente com imagem real ou simbolo claro de Conserto De Televisao, composicao limpa, beneficio em destaque e CTA visivel.
- Pouco texto e leitura rapida.
