# Inteligencia do Ramo

Esta pasta descreve o ramo individual `marketing_visual_empresas`.

Ela nao agrupa outros nichos. O foco aqui e entender quem compra marketing visual para empresas, quais dores esse publico sente, quais objeções aparecem e como comunicar valor de forma clara.

Este conhecimento podera orientar prompts, ofertas, textos, CTAs, validacoes e aprendizado futuro deste ramo especifico.
