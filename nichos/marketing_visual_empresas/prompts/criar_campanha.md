# Prompt: Criar Campanha

Prompt ideal futuro para criar campanha do nicho `marketing_visual_empresas`.

## Instrucao

Crie uma campanha para vender marketing visual para empresas.

A campanha deve ter:

- dor principal;
- promessa clara;
- mensagem da arte;
- CTA;
- sugestao visual;
- canal recomendado;
- cuidado para nao prometer resultado garantido.

## Saida esperada

- Nome da campanha.
- Publico-alvo.
- Texto principal.
- Texto de apoio.
- CTA.
- Direcao visual.
- Variações para Instagram e WhatsApp.
