# Prompt: Validar Arte

Prompt ideal futuro para validar uma arte do nicho `marketing_visual_empresas`.

Nao substitui validacoes atuais de producao.

## Instrucao

Avalie se a arte esta adequada para vender ou divulgar marketing visual para empresas.

Verifique:

- A mensagem principal e entendida rapidamente.
- O texto esta curto e legivel.
- O CTA esta claro.
- O visual transmite profissionalismo.
- A logo foi respeitada.
- Nao existem dados inventados.
- A arte fala com empresario ou dono de negocio.
- A composicao nao esta poluida.
- O resultado parece pronto para postar.

## Saida esperada

- `aprovada`: sim ou nao.
- pontos fortes.
- problemas encontrados.
- sugestoes de melhoria.
- alertas ao usuario, se necessario.
