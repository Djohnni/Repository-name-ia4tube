# Prompt: Criar CTA

Prompt ideal futuro para gerar CTAs do nicho `marketing_visual_empresas`.

## Instrucao

Crie CTAs para uma campanha de marketing visual para empresas.

O CTA deve ser curto, claro e voltado para acao imediata.

Priorize empresarios que precisam divulgar melhor o negocio.

## Tipos de CTA

- Rapidez.
- Profissionalismo.
- WhatsApp.
- Criacao de arte.
- Divulgacao de oferta.
- Presenca digital.

## Saida esperada

Retorne de 5 a 10 CTAs, separados por intencao.
