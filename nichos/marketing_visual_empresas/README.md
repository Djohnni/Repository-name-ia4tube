# Nicho: marketing_visual_empresas

Esta e a primeira pasta piloto de nicho.

O objetivo desta estrutura e organizar conhecimento especifico do nicho `marketing_visual_empresas`, incluindo prompts, regras, produtos, estrategias de marketing, memoria operacional e contratos esperados para pedidos futuros.

Este nicho representa marketing visual para empresas: criacao de artes, posts, imagens comerciais e materiais de divulgacao para negocios.

Importante: esta pasta ainda nao esta conectada ao runtime do sistema. O site, o app Android, o `server.js`, os runners e os prompts atuais continuam funcionando como antes.

Esta estrutura serve como base interna para evoluir uma arquitetura por nichos, sem alterar o fluxo atual de pedidos nesta etapa.
